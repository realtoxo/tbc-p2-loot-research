# SSC/TK Loot Council Research

Research pack for TBC Classic Phase 2 SSC/TK loot council decisions, synthesized from Zatar's loot-priority video, Knot's SSC/TK discussions, class-specialist videos, Wowhead references, and local transcript notes.

The focus is Tier 5 tokens, weapons, trinkets, offpieces, set timing, long-lived items, later swap value, and opportunity cost.

This is not a personal BiS list. The docs are written for an LC that needs to decide who gets a drop first, whether the player can actually equip it, how long the item lasts, and what the raid loses by giving it to one spec over another.

## LC Conventions

| Convention | Meaning |
|---|---|
| Protected set objective | A set bonus or slot plan that LC should deliberately finish, such as Mage 2p, Rogue 4p, Hunter 4p, Enhance 4p, Holy Priest 4p, Boomkin/Shadow 4p, or tank tier routing. |
| Equip constraint | A reason a player may not actually wear the item yet: Beast Lord 4p, Enhancement T4 2p, Spellfire/T5 routing, crafted chest, engineering helm, crush-immunity, or a better offset. |
| Longevity flag | The item or set remains valuable beyond the immediate lockout, lasts multiple phases, or becomes a later hit/crit/threat/tank swap. |
| Opportunity cost | The raid-level loss created by assigning a drop to one spec instead of the spec that has fewer alternatives or keeps the item longer. |
