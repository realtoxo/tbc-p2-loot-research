## LC Operating Rules

1. Award tokens by set plan, not by isolated BiS claims. A weak single piece can be correct if it completes a major set bonus.
2. Separate immediate-equip pieces from banked set pieces. If a player will not equip the token until 4p, they should only receive it when LC is deliberately pushing that player to the set.
3. Mage 2p, Rogue 4p, Hunter 4p, and certain tank/healer set plans are the early token anchors. Do not flatten every token into "biggest upgrade."
4. Do not break stronger old bonuses by accident. Hunter Beast Lord 4p, Enhancement T4 2p, and Prot Paladin T4 chest/shoulder threat setups are the highest-risk traps.
5. Track offpiece pressure. Rogue tier chest creates shoulder pressure; Warlock tier reduces pressure incorrectly when Warlock should be paid through offpieces; Holy Priest chest token fights a crafted chest that is already good.
6. Mark long-lived items. If a set bonus or token piece lasts into the next phase or becomes a later swap, it deserves more LC weight than a same-week sidegrade.

## Set Timing Archetypes

| Archetype | Specs | LC behavior |
|---|---|---|
| Immediate 2p | Arcane Mage | Give every core Mage 2p before finishing most other Hero-token projects. |
| Batch to 4p | BM Hunter, Rogue, Enhance, Boomkin, Shadow Priest, Feral Cat, Prot Warrior | Avoid spreading one-off pieces. Pick a player and finish the set when drops allow. |
| Strong individual pieces | Prot Paladin legs/gloves, Feral Bear pieces, Holy Priest pieces, Prot Warrior chest, Elemental chest/legs | Can be awarded even without immediate set completion if the piece is actually equipped. |
| Offpiece-funded specs | Warlock, Fury Warrior, Arms Warrior, Ret Paladin, many healers | Tier should be lower priority unless the exact slot is a real equip. Pay these specs through their high-impact offpieces. |

## Longevity and Swap Flags

| Spec/item path | Duration flag | LC meaning |
|---|---|---|
| Arcane Mage 2p T5 | Multi-phase / long-lived while playing Arcane | This is not just a Phase 2 opener. Protect the first two Hero tokens per Mage because the set bonus can remain valuable deep into later gearing. |
| Rogue 4p T5 | Multi-phase bridge into T6 | Rogue 4p is a major throughput set. Prioritize the correct four pieces rather than scattering Champion tokens. |
| BM Hunter 4p T5 | Multi-phase bridge until a real T6 set swap | Hunters often hold Beast Lord until 4p T5 is ready, then hold 4p T5 until a later set threshold beats it. Tokens that complete 4p are worth more than normal one-slot upgrades. |
| Enhance 4p T5 | Phase 2 core, later replaced by stronger T6 thresholds | Do not give random Enhance singles if they break T4 2p too early. Finish the correct 4p path. |
| Boomkin 4p T5 | Set-bonus build piece | Boomkin value comes from the 4p package and crit/proc setup, not from each isolated item. It can remain relevant until later pieces collectively replace it. |
| Shadow Priest 4p T5 | Set-bonus package with weak singles | Trickle lower-contested Defender pieces, then complete 4p deliberately. Do not overpay for helm unless it is the cleanest completion path. |
| Holy Priest 4p T5 | Strong Phase 2 healer set | Individual pieces are good, and Tier 5 encounters increase Circle of Healing relevance. Use Primal Mooncloth chest when available so the Kael Defender chest can go to tanks/Feral/Boomkin/Shadow first. |
| Prot Warrior T5 chest/4p | Tank set / encounter swap | Chest is a major piece. The 4p can be part of a threat or mitigation set, but helm/glove choices depend on engineering and Royal Gauntlets. |
| Prot Paladin T5 helm/legs/gloves | Tank set / threat pieces | Treat legs and gloves as real priorities, and helm as a route-dependent tank piece. T4 chest and shoulders can remain correct because of block value and the old threat bonus, so do not force Champion chest/shoulders unless the set math proves it. |
| Feral Bear T5 pieces | Tank set / encounter swap | Bear pieces are real tank gear and can remain in mitigation sets. Check whether the player is main bear or primarily cat before giving helm-level priority. |
| Mage T5 gloves | Temporary activator / later offset swap | Gloves can activate 2p, but final Mage set often wants `Gauntlets of the Sun King` as offset. This is still fine if gloves are the fastest path to 2p. |
| Rogue T5 chest | Avoid unless drop pattern forces it | Tier chest pushes Rogue toward `Shoulderpads of the Stranger`. Default route is tier helm/shoulders/gloves/legs plus `Bloodsea Brigand's Vest`. |
| Warlock T5 pieces | Mostly short-lived or backup | Warlock longevity comes from offpieces, not Hero tokens. Do not treat Warlock tier as compensation for missing Vashj chest/wand/weapon. |
| Resto Druid spirit offpieces | Healer equity path | Resto Druid gets real value from spirit through Tree of Life aura and Innervate. Use spirit cloak/rings/weapons as compensation when Defender tokens are going to higher-impact set plans. |

## Champion Token: Paladin, Rogue, Shaman

### Slot Priority

| Slot | Recommended priority | Equip constraint |
|---|---|---|
| Gloves, Leotheras | Rogue > Prot Paladin > Enhance > biggest upgrade | Rogue wants fast 4p and can equip as part of that path. Prot Paladin can move above Rogue if Rogue cannot realistically finish 4p soon. Enhance mostly wants gloves as a 4p piece, not as a standalone break from T4. |
| Legs, Karathress | Prot Paladin, unless Rogue completes 4p that week > Elemental > Enhance if completing 4p > Holy Paladin > Resto Shaman > Ret | Prot Paladin legs are an elite immediate equip. Rogue legs jump only when they finish 4p. Elemental legs are a real individual piece. Holy Paladin generally values tier legs before Resto Shaman because Resto has later mail alternatives. |
| Helm, Vashj | Rogue > Enhance > Prot if this is the planned tank route > Ret/Holy/Resto > Elemental | Rogue helm is part of the preferred 4p. Enhance uses helm in its 4p, but should often hold T4 head/shoulder until the final swap. Prot can move up when helm completes a real uncrushable/threat set; otherwise check engineering goggles. Elemental should usually keep T4 helm. |
| Shoulders, Void Reaver | Rogue > Holy Paladin > Resto Shaman > Prot > others | Rogue shoulders are central because they prevent the bad tier-chest/off-shoulder route. Prot Paladin often keeps T4/Justicar shoulders for block and set math. Enhancement and Ret should usually target `Shoulderpads of the Stranger`, not Champion shoulders. |
| Chest, Kael'thas | Enhance > Resto/Elemental/Holy > Ret/Rogue/Prot | Enhance needs chest for the standard 4p and it is a strong piece. Rogue should avoid tier chest by default. Prot and Ret have weaker set logic here. |

### Class Plans

| Spec | Target plan | Constraints and swaps |
|---|---|---|
| Rogue | 4p Deathmantle: helm, shoulders, gloves, legs; offset `Bloodsea Brigand's Vest` | Highest Champion set value. Avoid tier chest unless your drop pattern makes Bloodsea unavailable and shoulders uncontested. This set is a multi-phase bridge, so finish one Rogue rather than giving every Rogue one token. |
| Enhancement Shaman | 4p Cataclysm: helm, chest, gloves, legs; avoid shoulders | The 4p haste package is the target. Do not break T4 head/shoulders early for random T5 singles unless the full swap is ready or the single piece is clearly equipped without losing the old set value. |
| Prot Paladin | Legs first, gloves high, helm if it is the planned tank route; shoulders/chest low | Legs and gloves are legitimate tank/threat pieces. Helm can be strong, but depends on engineering goggles and set math. Chest and shoulders are usually low because T4 chest/shoulders preserve block value and the old threat bonus. Confirm crush-immunity, defense, stamina, and spellpower before calling the token an upgrade. |
| Elemental Shaman | Chest and legs; maybe gloves if badly geared | Set bonuses are weak. Keep T4 head/shoulders in most cases. Do not let Elemental consume helm/shoulder tokens before Rogue/Enhance set plans. |
| Holy Paladin | Head/shoulder/chest as medium priority | No major set-bonus urgency. Award when it is a true equip or healer-compensation piece after the major Champion plans are moving. |
| Resto Shaman | Head/shoulder/chest as medium-low priority | Gloves and legs have offset competition. Resto Shaman should receive healer equity, but not at the cost of Rogue 4p or critical Prot Paladin legs. |
| Ret Paladin | Helm/chest only if useful; generally low token priority | Ret does not have a major T5 set objective. Use weapons, belts, shoulders, and trinkets for Ret value instead of forcing tier. |

## Defender Token: Warrior, Priest, Druid

### Slot Priority

| Slot | Recommended priority | Equip constraint |
|---|---|---|
| Gloves, Leotheras | Feral Cat/Bear > Holy Priest > Prot Warrior/Shadow/Boomkin by 4p proximity > Resto Druid > Fury/Arms | Feral and Holy Priest can equip cleanly. Prot/Shadow/Boomkin should be judged by whether this moves them toward 4p. DPS Warriors are low-priority offpiece specs. |
| Legs, Karathress | Bear > Holy Priest > Prot Warrior/Boomkin/Shadow by 4p proximity > Resto Druid > Fury/Arms | Bear legs are strong immediately. Holy Priest legs are strong. Boomkin/Shadow mostly want legs if they fit the planned 4p route. |
| Helm, Vashj | Main Bear > Balance/Holy Priest > Prot Warrior/Shadow only if committing to 4p > Fury/Arms > Resto Druid | Helm is heavily contested. Main Bear priority is real. Shadow Priest is usually steered away from helm to complete 4p with shoulders/chest/gloves/legs. Prot Warrior may avoid helm through engineering. |
| Shoulders, Void Reaver | Bear > Prot Warrior > Balance/Holy Priest > Shadow if finishing 4p > Resto/Fury/Arms | Bear and Prot Warrior have strong claims. Balance/Holy are next. Shadow rises when shoulders complete or materially advance 4p. |
| Chest, Kael'thas | Prot Warrior > Feral > Balance/Shadow working 4p > Resto Druid > Fury/Arms > Holy Priest | Prot Warrior chest is exceptional. Holy Priest should usually use Primal Mooncloth chest and complete 4p elsewhere. |

### Class Plans

| Spec | Target plan | Constraints and swaps |
|---|---|---|
| Feral Bear | Most pieces are real, especially helm/shoulders/legs/chest | Main Bear gets high Defender priority because these pieces can live in tank sets beyond the week they drop. If the player is a cat-first hybrid, lower helm priority because cat often keeps Wolfshead-style logic. |
| Feral Cat | 4p, usually shoulders/chest/gloves/legs; avoid helm | Cat wants the Shred set bonus. Do not award Bear-priority helm unless the player actually tanks enough to justify it. |
| Balance Druid | 4p, commonly helm/shoulders/chest/gloves with `Spellstrike Pants` or another leg offset | Boomkin's value is set-bonus plus crit/proc build, not individual tokens. Gloves can be a good low-drama fourth piece. Legs are optional because crafted/offpiece legs reduce pressure. |
| Holy Priest | 4p, usually helm/shoulders/gloves/legs with `Primal Mooncloth Robe` | Individual pieces are excellent. Chest token should be low because the crafted chest keeps Holy Priest functional while tanks/Feral/casters fight for Defender chest. |
| Shadow Priest | 4p, usually shoulders/chest/gloves/legs; avoid helm | Individual pieces are not great, so do not over-prioritize early singles. The right play is to pick up less-contested tokens and then finish 4p when close. |
| Prot Warrior | Wants chest and may want 4p; helm/glove route depends on gear | `Destroyer Chestguard` is a major tank piece. Engineering helm can free the Vashj helm token. `Royal Gauntlets of Silvermoon` can be excellent but may force tier helm, so check the whole set before assigning gloves/helm. |
| Fury Warrior | Usually no tier except maybe helm as a filler | Defender tokens are too contested to prioritize Fury. Fury should be served through weapons and physical offpieces unless tokens are rotting. |
| Arms Warrior | Usually no tier project; treat as weapon/trinket/offpiece spec | The reviewed Arms transcript supports stat and weapon-specialization context, not a Tier 5 objective. Only award Defender tier if it is an immediate equip after protected set users are handled. |
| Resto Druid | Mainly helm/shoulders/chest; gloves/legs are lower-value sustain pieces | Resto Druid can use tier in Phase 2, but the set does not outrank Bear, Holy Priest, Boomkin, Shadow, or Prot Warrior set timing in most slots. |

## Hero Token: Hunter, Mage, Warlock

### Slot Priority

| Slot | Recommended priority | Equip constraint |
|---|---|---|
| Gloves, Leotheras | Mage > Hunter > Warlock | Mage gloves can be the fastest path to 2p. Hunter gloves are for the 4p batch. Warlock generally does not want this slot. |
| Legs, Karathress | Mage > Survival Hunter or Hunter with real upgrade > Warlock | Mage first. BM Hunter usually uses `Leggings of Murderous Intent` as offset, but Survival may prefer tier legs. Warlock legs are mostly a backup/stamina piece. |
| Helm, Vashj | Mage > Hunter > Warlock | Mage and Hunter set bonuses matter more. Warlock can move up only if Hunters are not being pushed to 4p and the Warlock will actually equip helm. |
| Shoulders, Void Reaver | Mage > Hunter > Warlock | Mage 2p/4p and Hunter 4p are the real goals. Warlock shoulders are not a priority item. |
| Chest, Kael'thas | Mage > Hunter > Warlock | Mage chest supports the T5 route. Hunter chest can be part of the 4p batch. Warlock should normally get `Vestments of the Sea-Witch` instead. |

### Class Plans

| Spec | Target plan | Constraints and swaps |
|---|---|---|
| Arcane Mage | First goal: 2p immediately. Later: 4p, often helm/shoulders/chest/legs with `Gauntlets of the Sun King` offset | Mage 2p is the strongest early Hero-token objective and can remain valuable across later gearing. Gloves are acceptable if they activate 2p quickly, but may later become the offset slot. Do not give Mage Vashj cloth chest over Warlock. |
| BM Hunter | 4p Rift Stalker only when complete; usually offset legs with `Leggings of Murderous Intent` | Do not break Beast Lord 4p for isolated T5 pieces. Bank tokens until the Hunter can swap into 4p. Once Mage 2p is solved, stack one Hunter to 4p instead of spreading unusable singles. |
| Survival Hunter | 4p can include tier legs; weapon/stat-stick decisions are separate | Survival may value tier legs more than BM because the offpiece plan differs. Confirm whether the Hunter is actually maintaining Survival/Expose Weakness before prioritizing Survival-specific swaps. |
| Warlock | Helm optional; chest only if Vashj chest is delayed; legs are a backup/stamina piece; no shoulders/gloves | Warlock is low Hero-token priority because their real long-lived pieces are offpieces: `Vestments of the Sea-Witch`, `Wand of the Forgotten Star`, `Fang of the Leviathan`, `Fathomstone`, boots, and rings. Tier should not be used as a substitute for those. |

## Equip Constraint Checklist

Ask these before assigning any token:

1. Does this token complete 2p or 4p this week?
2. If it does not complete a bonus, will the player equip it immediately?
3. Does equipping it break a stronger old bonus, especially Beast Lord 4p, Enhancement T4 2p, or Prot Paladin T4 chest/shoulder routing?
4. Is the player being routed into a worse offpiece conflict, such as Rogue tier chest forcing `Shoulderpads of the Stranger`?
5. Is there a crafted/offpiece alternative that should change priority, such as `Primal Mooncloth Robe`, `Spellstrike Pants`, engineering helms, `Bloodsea Brigand's Vest`, `Leggings of Murderous Intent`, or `Gauntlets of the Sun King`?
6. Is the piece long-lived into later phases or useful as a hit/crit/defense/resist/threat swap?
7. If the token is being banked, has LC committed to finishing that player soon, or is the raid just creating dead inventory?

## High-Risk Misallocations

| Mistake | Why it hurts |
|---|---|
| Delaying Mage 2p to finish lower-impact Hero-token projects | Arcane Mage 2p is a major raid-DPS unlock and can remain relevant for later gearing. |
| Giving BM Hunters isolated T5 singles | The Hunter may not equip them because Beast Lord 4p stays intact until 4p Rift Stalker is ready. |
| Giving Rogue tier chest by default | It consumes the Kael Champion chest and pushes Rogue toward contested offset shoulders. |
| Treating Warlock Hero tokens as compensation | Warlock power comes from offpieces. If locks are missing Vashj chest, wand, Fang, or Fathomstone, tier is the wrong payment. |
| Giving Enhancement shoulders | The normal Enhance 4p route avoids shoulders and uses stronger physical offset shoulders. |
| Giving Prot Warrior helm without checking engineering/Royal Gauntlets route | Tank sets are conditional. A helm token can be saved if engineering helm solves the set. |
| Giving Holy Priest Defender chest early | `Primal Mooncloth Robe` keeps Holy Priest strong while Defender chest goes to specs with fewer clean alternatives. |
| Giving Fury/Arms Warrior Defender tier ahead of set users | DPS Warrior Phase 2 value comes from weapons, trinkets, and physical offpieces, not Tier 5. |
