This is the boss-by-boss table version of the research pack. It is intended for loot council calls, not as a personal BiS sheet.

## How To Read These Tables

| Term | Meaning |
|---|---|
| `>` | Preferred order when players are similarly reliable and the item is a real equip. |
| `=` | Same priority lane; break ties by attendance, performance, current gear, or planned set route. |
| `Biggest upgrade` | No strong spec-first claim in the current research; award to the player who can equip it now and keep it longest. |
| `Temp` | Useful bridge item, but do not let it beat a protected set bonus, unique weapon, or long-lived offpiece. |
| `Avoid` | Do not award while real candidates exist unless your roster has a special reason. |

For all tokens, protected bonus completion beats isolated BiS. Do not award banked pieces unless LC is deliberately finishing that set soon. The recurring protected objectives are Mage 2p, Rogue 4p, Hunter 4p, Enhancement 4p, main tank set pieces, Holy Priest 4p, and Boomkin/Shadow 4p when close.
